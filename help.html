Bitbucket Query Language | Help

This extension helps you to navigate through Bitbucket repositories with ease.
You can use the commands to open the repositories, branches, tags, commits, pull requests, pipelines, deployments, and compare branches or tags.

Type open to start querying Bitbucket. If you new to BQL, please go through the list of commands below.

Command Description Url
Set workspace-name Set the active workspace -
List workspace-name Open the list of repositories in the workspace bitbucket.org/workspaceName/workspace/repositories
repository-name Open the repository default source page bitbucket.org/workspaceName/repositoryName/overview
repository-name BRANCH Open the branches of the repository bitbucket.org/workspaceName/repositoryName/branches
repository-name BRANCH branch-name Open the source page of the branch bitbucket.org/workspaceName/repositoryName/src/branch-name
repository-name BRANCH branch-name COMMIT Open the commit history of the branch bitbucket.org/workspaceName/repositoryName/commits/branch/branch-name
repository-name BRANCH branch-name COMMIT commit-id Open the commit details bitbucket.org/workspaceName/repositoryName/commits/commit-id
repository-name TAG tag-name Open the source page of the tag bitbucket.org/workspaceName/repositoryName/src/tag-name
repository-name TAG tag-name COMMIT Open the commit history of the tag bitbucket.org/workspaceName/repositoryName/commits/tag/tag-name
repository-name TAG tag-name COMMIT commit-id Open the commit details bitbucket.org/workspaceName/repositoryName/commits/commit-id
repository-name COMMIT Open the commit history of the repository bitbucket.org/workspaceName/repositoryName/commits
repository-name COMMIT commit-id Open the commit details bitbucket.org/workspaceName/repositoryName/commits/commit-id
repository-name PR Open the pull requests of the repository bitbucket.org/workspaceName/repositoryName/pull-requests
repository-name PR pull-request-no Open the pull request details bitbucket.org/workspaceName/repositoryName/pull-requests/pull-request-no
repository-name PIPELINE Open the pipelines of the repository bitbucket.org/workspaceName/repositoryName/pipelines
repository-name PIPELINE pipeline-no Open the pipeline details bitbucket.org/workspaceName/repositoryName/pipelines/results/pipeline-no
repository-name DEPLOY Open the deployments of the repository bitbucket.org/workspaceName/repositoryName/deployments
repository-name DEPLOY environment-name Open the deployment details bitbucket.org/workspaceName/repositoryName/deployments/environment-name
repository-name COMPARE from-branch-or-tag TO to-branch-or-tag Compare branches or tags bitbucket.org/workspaceName/repositoryName/branches/compare/from-branch-or-tag%0Dto-branch-or-tag
repository-name DIFF branch-name-or-tag Diff branch or tag to the default branch or tag bitbucket.org/workspaceName/repositoryName/branch/branch-name-or-tag
repository-name DIFF from-branch-or-tag TO to-branch-or-tag Diff branches or tags bitbucket.org/workspaceName/repositoryName/branch/from-branch-or-tag?dest=to-branch-or-tag
